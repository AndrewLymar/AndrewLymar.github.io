armoredcase.com
HTML | CSS | Bootstrap | JavaScript | AJAX | jQuery | PHP | MySql

Интернет-магазин написан мной полностью с нуля, без использования CMS, адаптивный, полностью динамический (при этом seo-friendly). Писал как на чистом javascript, так и на Jquery, т.к. сайт с динамической загрузкой контента, то часто использовал AJAX, смена url без перезагрузки страницы реализована за счет HTML5 History API. Сайт адаптирован под все возможные устройства, при верстке использовал Bootstrap. На стороне серевера использовал PHP, как для написания самого сайта, так и для админки, в качестве БД использовал MySql.
